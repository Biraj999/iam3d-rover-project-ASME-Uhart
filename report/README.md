# IAM3D Rover Project

University of Hartford ASME team project for the IAM3D Rover Challenge.

This project involved the design, fabrication, wiring, programming, and testing of a fully 3D-printed remote-controlled FPV rover. Our team developed the rover under a tight timeline and successfully earned **2nd place** in competition.

## Overview

The IAM3D Rover Project was created as a multidisciplinary engineering build combining mechanical design, 3D printing, embedded programming, electronics integration, wireless remote control, and FPV operation.

## Highlights

- Fully 3D-printed rover structure
- Tracked drive system
- ESP32-based embedded control
- FlySky remote control integration
- FPV display for real-time navigation
- Robotic arm mechanism
- 2nd place competition finish

## My Contribution
